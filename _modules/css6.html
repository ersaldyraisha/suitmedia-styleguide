## No ID and !important in CSS Jangan ada deklarasi ID ataupun !important di dalam CSS karena ID memiliki tingkat [specificity yang tinggi](http://www.smashingmagazine.com/2007/07/27/css-specificity-things-you-should-know/). Apabila memang terpaksa harus menggunakan, gunakan attribute selector. Ini adalah sebuah hack dan sebisa mungkin jangan sampai ada. Penggunakan `!important` hanya diperbolehkan untuk override style third party ataupun style plugin. ``` /* NO! */ #myElem { prop: value; } button { padding: .5em 1em !important; } /* YES! */ [id="myElem"] { prop: value; } button { padding: .5em 1em !important; } ```
