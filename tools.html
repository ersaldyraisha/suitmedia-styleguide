§ Slick Carousel

Digunakan apabila membutuhkan image slider ataupun carousel.

Documentation
kenwheeler.github.io/slick/
Version
1.5.3
CLI
bower i slick-carousel#1.5.3 --save
Bower registry
"slick-carousel": "~1.5.3"
Note

Slick menyediakan 2 file css, slick.css dan slick-theme.css. Disarankan hanya include slick.css dan membuat style sendiri sesuai kebutuhan masing-masing project.
