Avoid deep nesting

Maksimal nesting adalah 3 level, lebih dari itu hindari. Nesting level paling akhir apabila adalah Pseudo Class atau Pseudo Element diperbolehkan.

// Bad
ul {
    li {
        a {
            span {
                ...
            }
        }
    }
}

// Good
ul {
    li {
        a {
            ...
        }
    }
}

ul {
    li {
        a {
            &:hover {
                ...
            }
        }
    }
}
