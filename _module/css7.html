Avoid contextual styles

Intinya jangan buat style yang medeskripsikan untuk apa atau untuk siapa style itu ditujukan.

Styles harus dibuat se-abstract mungkin agar dapat dipakai berkali-kali dan dapat dipakai untuk segala konteks.

/* NO! */
.btn-download {
    prop: value;
}

/* YES! */
.btn-primary {
    prop: value;
}

/* NO! */
.faq-wrapper {
    width: 400px;
}

/* YES! */
.container {
    width: 800px;
}

.container--small {
    width: 400px;
}
