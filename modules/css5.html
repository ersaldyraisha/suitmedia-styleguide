## CSS Prefixing CSS prefixing di-handle oleh autoprefixer, jadi tidak perlu menambahkan vendor-prefix secara manual atau menggunakan mixin. Perlu diketahui autoprefixer hanya meng-parse **CSS property official yang telah ditetapkan oleh W3.**. Jadi property `appearance` apabila memang ingin digunakan, harus input vendor prefix secara manual. Khusus gradient, ikuti [deklarasi yang telah ditetapkan oleh W3](//developer.mozilla.org/en-US/docs/Web/CSS/linear-gradient) ``` .box { background-image: linear-gradient(to bottom, #fff, #eee); transform: scale(1); transition: all .3s ease; } ```
