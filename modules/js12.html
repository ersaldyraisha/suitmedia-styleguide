## Loop Untuk iterasi element, gunakan `for` daripada jQuery `each` karena `for` lebih cepat dari segi performance. [Reference](http://jsperf.com/jquery-each-vs-for-loop/39) ``` // slow var $elems = $('.elem'); $elems.each( function (i, el) { var $this = $(this); console.log($this); }); // fast var $elems = $('.elem'); for ( var i = 0, j = $elems.length; i < j; i++ ) { var curr = $elems.eq(i); console.log(curr); } ```
