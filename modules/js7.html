## Gunakan function expression Dalam deklarasi sebuah function, gunakan function expression daripada function declaration. ``` // bad function move() { // code } // good var move = function () { // code }; ``` Format deklarasi function expression dapat dilihat di samping. Spasi sebelum dan sesudah buka tutup kurung, dan jangan lupa tambahkan titik koma setelah tutup kurung kurawal. ``` var move = function () { // ... }; ```
