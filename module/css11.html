## Don't use CSS @import
Alasan kenapa jangan menggunakan CSS `@import` adalah: * File CSS yang dipanggil via @import tidak di cache di browser. * File CSS yang dipanggil via @import tidak dapat didownload oleh browser secara paralel. * Mengurangi performance web.
``` /* NO! */ @import url('hello.css'); ``` ``` <!-- YES! --> <link rel="stylesheet" href="hello.css"> <!-- or use scss import --> ```
