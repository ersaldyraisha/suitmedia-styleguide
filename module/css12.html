## Use transform and opacity for high performance animation
Untuk membuat animasi contoh pada saat image di hover, image membesar, jangan gunakan width dan height. Gunakan property `transform` agar dapat dirender menggunakan GPU, bukan CPU. Pada dasarnya, untuk mendapatkan high performance animation, gunakan: * Translate * Scale * Rotate * Opacity
```css /* NO! */ div { position: relative; top: 0; transition: all .3s ease; &:hover { top: 100%; } } /* YES! */ div { trasnform: translateY(0); transition: all .3s ease; &:hover { transform: translateY(100%); } } ```
