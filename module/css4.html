## Spacing and new lines
Agar lebih mudah dibaca dan maintainable, tambahkan space dan new line secukupnya.
```css // Bad a{ prop: value; &:hover,&:focus{ prop: value; } &:active{ prop: value; } span{ color: #999; } } button{ ... } // Good a { prop: value; &:hover, &:focus { prop: value; } &:active { prop: value; } span { color: #999; } } button { ... } ```
