## Convert String to Int Untuk convert string ke int, gunakan Math.floor daripada parseInt karena Math.floor lebih baik dari segi performance. [Reference](http://jsperf.com/math-floor-vs-math-round-vs-parseint/148) ``` var value = '1000'; console.log(typeof value); // String var newValue = Math.floor(value); console.log(typeof newValue); // Number ```
