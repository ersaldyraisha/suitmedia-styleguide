## Standard HTML Declaration
Aturan dalam deklarasi HTML * Gunakan kutip 2, bukan kutip 1. * Urutan penempatan attribute: 1. Class 2. ID 3. Attribute bawaan (href, src) 4. Attribute global 5. Data attribute * Apabila deklarasi tag sudah terlalu panjang, diperbolehkan untuk menambahkan new line untuk tiap attribute.
```css <!-- Bad --> <a data-attribute="link" href="..." id='link' class="link" title="my link">Link</a> <!-- Good --> <a class="link" id="link" href="..." title="my link" data-attribute="link">Link</a> <a class="link" id="link" href="..." title="my link" data-attribute="link">Link</a> ```
