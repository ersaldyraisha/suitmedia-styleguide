## Hindari anonymous function
Buatlah sebuah function dan attach function tersebut ke handler yang dibutuhkan. Anonymous function membuat code tidak reusable dan menambahkan penggalan code yang ter-indent. Dengan tidak membuat anonymous function juga menambah readability dan menjelaskan tujuan fungsi yang dijalankan.
``` // bad btn.click( function (e) { // Tidak tau apa yang terjadi apabila class 'bar' ditambahkan foo.addClass('bar'); e.preventDefault(); }); // good var openMenu = function (e) { foo.addClass('bar'); e.preventDefault(); }; btn.click( openMenu ); ```
