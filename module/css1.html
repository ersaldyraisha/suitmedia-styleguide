## CSS Declaration
Urutan deklarasi properti CSS adalah: 1. Mixin 2. Position 3. Box model 4. Type 5. Visual 6. Misc 7. Pseudo element 8. Pseudo class 9. Media Queries
```css selector { @include thing1; @include thing2; // Position position: relative; top: 0; right: 0; bottom: 0; left: 0; z-index: 0; // Box model display: block; width: 100px; height: 100px; padding: 1em; border: solid 1px #222; margin: 0 auto; // Type font: 400 16px sans-serif; line-height: 1.6em; text-transform: uppercase; color: #222; // Visual background: #eee; box-shadow: inset 0 0 2px #555; cursor: auto; opacity: 1; transform: scale(1); // Misc transition: all .3s ease; // Pseudo element &:before, &:after { ... } // Pseudo class &:first-child { ... } &:hover { ... } // Media Queries @media ... { ... } } ```
