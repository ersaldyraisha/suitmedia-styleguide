## Penamaan class dan ID
Penamaan class dan ID hanya diperbolehkan menggunakan: * Alphabet a-z || A-Z * Numeric 0 - 9 * Dash '-' * Underscore '_' * Harus berawalkan alphabet Aturan penamaan untuk: * Class: tidak boleh camelcase dan harus lowercase (kecuali bawaan plugin) * ID: harus camelcase
