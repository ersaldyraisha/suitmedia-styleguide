## Simpan string yang dibutuhkan dalam Object
Ada case di mana kita harus mengecek apakah sebuah element mempunyai nama class atau tidak. Dan terkadang kita harus mengecek apakah sebuah element memiliki class-class yang ada. Untuk mempermudah, string class dapat disimpan ke dalam object litaral.
``` // ok var classActive = 'is-active', classIdle = 'is-idle', classLabel = 'label'; var checkElement = function ( elem ) { if ( !elem.hasClass( classLabel ) ) return; if ( elem.hasClass( classActive ) ) { elem.removeClass( classActive ); elem.addClass( classIdle ); } }; // better var classes = { active: 'is-active', idle : 'is-idle', label : 'label' }; var checkElement = function ( elem ) { if ( !elem.hasClass( classes.label ) ) return; if ( elem.hasClass( classes.active ) ) { elem.removeClass( classes.active ); elem.addClass( classes.idle ); } }; ```
