## Label untuk tiap input Secara semantic dan accessibility, tiap input harus memiliki label, meskipun secara design label digantikan oleh placeholder. #### Contoh: Form tanpa label ``` <div class="form-row"> <!-- class sr-only untuk menyembunyikan label, namun tetap accessible melalui screen reader --> <label class="sr-only" for="exForm1">Username</label> <input class="form-input" id="exForm1" type="text" placeholder="Username"> </div> <div class="form-row"> <label class="sr-only" for="exForm2">Password</label> <input class="form-input" id="exForm2" type="password" placeholder="Password"> </div> ```
